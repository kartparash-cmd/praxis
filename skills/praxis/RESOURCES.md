# Resources

Public seed of high-trust sources so praxis honors its own "never trust
parametric knowledge" rule out of the box. Add your own stacks here (or in a
private overlay). Format: see `reference/RESOURCES-FORMAT.md`.

## Spaced repetition / learning science
- [Make It Stick (Brown, Roediger, McDaniel)](https://www.retrievalpractice.org/) — testing effect, spacing. Tier: secondary.
- [Augmenting Long-term Memory (Nielsen)](https://augmentingcognition.com/ltm.html) — practical spaced repetition. Tier: secondary.

## FastAPI
- [FastAPI docs](https://fastapi.tiangolo.com/) — official. Tier: primary.
- [Starlette docs](https://www.starlette.io/) — the ASGI layer under FastAPI. Tier: primary.

## pgvector
- [pgvector README](https://github.com/pgvector/pgvector) — official; index types, operators. Tier: primary.
- [Postgres: HNSW vs IVFFlat](https://github.com/pgvector/pgvector#indexing) — official indexing guidance. Tier: primary.

## Retrieval-augmented generation (RAG)
- [Anthropic: contextual retrieval](https://www.anthropic.com/news/contextual-retrieval) — chunking + retrieval quality. Tier: primary.
